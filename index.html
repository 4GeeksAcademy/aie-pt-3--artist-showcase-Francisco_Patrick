About

Diseño para historias que viven sobre la mesa

Mi enfoque combina claridad funcional con impacto visual. Cada carta, icono y tablero debe contar una historia sin romper el ritmo de juego. Trabajo con paletas vibrantes, jerarquía tipográfica y sistemas modulares para que cada componente sea jugable y memorable.

  • Identidad visual de mundos RPG
  • Sistemas de iconografía y UI física
  • Diseño editorial de manuales y bestiarios
  • Prototipado visual para playtesting
